# Forest Briggs Portfolio

Welcome to my personal portfolio website! This site showcases my skills, experience, and projects as a software engineer. The goal is to provide a space where recruiters and potential clients can learn more about me and get in touch.

## 🔗 Live Site
You can check out the live version of my portfolio [here](https://forestbriggsdev.vercel.app/).

## 🚀 Features

- **Contact Form:** Reach out directly through the contact form, which sends me an email and sends an automatic confirmation reply.
- **Resume Download:** Download a copy of my resume directly from the site.
- **Interactive Design:** The site uses a starry night Lottie animation for a modern and engaging visual experience.
- **Responsive Layout:** The design is fully responsive, making it accessible on all devices.

## 🛠️ Technologies Used

- **Frontend:** Next.js, React, TypeScript
- **Styling:** Tailwind CSS
- **Email Integration:** EmailJS (for handling form submissions and auto-replies)
- **Animations:** Lottie Files (for the starry night background)

## 📧 Contact Form Setup

The contact form sends an email directly to me when submitted and automatically replies to the sender. This is handled using **EmailJS**.
